# Петрович × Горбилет — «Где лето лучше: в городе или на даче?»

Лендинг конкурса. Один самодостаточный файл — все изображения, шрифты и логика встроены внутрь `index.html`, внешних зависимостей нет (кроме Google Fonts).

## Публикация на GitHub Pages

1. Создайте репозиторий и загрузите в него `index.html` (и при желании этот `README.md`).
2. Откройте **Settings → Pages**.
3. В разделе **Build and deployment** выберите источник **Deploy from a branch**, ветку `main` и папку `/ (root)`.
4. Сохраните. Через минуту сайт будет доступен по адресу `https://<ваш-логин>.github.io/<имя-репозитория>/`.

Файл `index.html` открывается и локально — просто двойным щелчком в браузере.
